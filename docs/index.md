# TimeDoctorAlert — Документация проекта

> Сгенерировано: 2026-03-14 | Режим: initial_scan (exhaustive)

## Обзор проекта

**TimeDoctorAlert** — десктопная утилита для Windows, которая мониторит окна приложения Time Doctor и воспроизводит звуковой сигнал при их появлении. Работает в системном трее.

### Краткая справка

- **Тип:** Монолит (десктопное приложение)
- **Язык:** C# (.NET Framework 4.8)
- **UI-фреймворк:** WPF
- **Архитектурный паттерн:** Событийно-ориентированная polling-утилита
- **Точка входа:** `TimeDoctorAlert/App.xaml` → `MainWindow.xaml.cs`
- **Ключевые зависимости:** NAudio 2.2.1, Serilog 3.1.1, Serilog.Sinks.Seq 6.0.0

## Быстрый старт

1. Откройте `TimeDoctorAlert.sln` в Visual Studio 2017+
2. Убедитесь, что установлен .NET Framework 4.8 SDK
3. Соберите и запустите проект (F5)
4. Приложение появится в системном трее
