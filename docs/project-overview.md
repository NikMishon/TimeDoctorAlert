# TimeDoctorAlert — Обзор проекта

## Назначение

TimeDoctorAlert — кроссплатформенная консольная утилита (Windows, macOS, Linux), которая мониторит появление окон приложения Time Doctor и воспроизводит звуковой сигнал для привлечения внимания пользователя. На Windows приложение дополнительно работает в системном трее. Утилита автоматически отслеживает окна процессов "Time Doctor" и "timedoctor2" определённого размера (>550x100 пикселей).

## Краткое описание

Приложение решает задачу оперативного оповещения пользователя о появлении окна Time Doctor (например, запроса подтверждения активности). При обнаружении нового окна Time Doctor воспроизводится MP3-файл ("Имперский марш" в 8-bit версии) в цикле. Звук продолжается до тех пор, пока окно не будет закрыто или не пройдёт 1 минута.

- **Тип репозитория:** Монолит
